# Outfit Recommendation System 

An ML-based system that recommends outfits based on user preferences, weather, and occasion.

## Features
- Recommends top outfits based on user input
- Filters by occasion: Casual, Formal, Party, Sports
- Weather-based suggestions: Summer, Winter, Rainy
- REST API built with FastAPI

## Tech Stack
- **Backend**: Python, FastAPI, Uvicorn
- **ML Model**: Scikit-learn, Pandas, Numpy
- **Image Processing**: Pillow, OpenCV
- **Frontend**: HTML, CSS, JavaScript
